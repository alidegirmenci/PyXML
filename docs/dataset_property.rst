Dataset Property
================

The "Dataset property" tab allows both visual and numerical analysis 
of the dataset. Also in this tab, the importance of features can be 
computed using two different methods.  In this way, users can explore 
the features of the dataset and thus gain in-depth knowledge about the 
dataset. The default version of the "Dataset Property" tab is shown in 
Figure 22.
